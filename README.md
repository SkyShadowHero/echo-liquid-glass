# Echo Liquid Glass

为 EchoMusic 底部音乐控件添加 iOS 风格液态玻璃折射效果。

基于 SVG feDisplacementMap + 动态 Canvas 位移贴图，仅折射（可叠加模糊）。

## 依赖

需要配合 [echo-miuix-plugin](https://github.com/SkyShadowHero/echo-miuix-plugin) 使用，提供 MIUIX 主题颜色变量。

## 效果实现

液态玻璃折射效果实现自 [liquid-glass](https://github.com/archisvaze/liquid-glass)。
